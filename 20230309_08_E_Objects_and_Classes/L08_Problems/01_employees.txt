20230309 - JavaScript Front-End 202302 - Objects and Classes - Exercises
01 - Employees - judge: https://judge.softuni.org/Contests/Compete/Index/3792#0


You're tasked to create a list of employees and their personal numbers.
You will receive an array of strings.
Each string is an employee name and to assign them a personal number you have to find the length of the name (whitespace included). 
Try to use an object.
At the end print all the list employees in the following format:
 "Name: {employeeName} -- Personal Number: {personalNum}" 


Test Data:
---------------------------


Input 1:
--------
[
'Silas Butler',
'Adnaan Buckley',
'Juan Peterson',
'Brendan Villarreal'
]


Output 1:
--------
Name: Silas Butler -- Personal Number: 12
Name: Adnaan Buckley -- Personal Number: 14
Name: Juan Peterson -- Personal Number: 13
Name: Brendan Villarreal -- Personal Number: 18


---------------------------


Input 2:
-------
[
'Samuel Jackson',
'Will Smith',
'Bruce Willis',
'Tom Holland'
]


Output 2:
--------
Name: Samuel Jackson -- Personal Number: 14
Name: Will Smith -- Personal Number: 10
Name: Bruce Willis -- Personal Number: 12
Name: Tom Holland -- Personal Number: 11
