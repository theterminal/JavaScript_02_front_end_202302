How can we add class to an element dynamically. The class already exists in our CSS Style sheet.

As it is visible from the example, the first part of the JS file takes all the 'p' elements of the page and adds #d9d9d9 background to each one of them.

The second part of the JS file takes only the first 'p' element from all the 'p' elements on the page and adds blue color and different font family to its text.

The HTML page does NOT contain any classes! All are added dynamically using JS.

The third part you must figure out on your own :)

You can add more than one class at a time. You can remove classes using yourElement.classList.remove() You can replace classes by using yourElement.classList.replace() ... and much more.
